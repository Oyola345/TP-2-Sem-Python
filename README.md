## Requisitos previos
* **Python 3.13.12**
* **Git y pip instalado**

## 🛠️ Instalación y comandos necesarios
1. Clonar el repositorio:
   ```bash
   git clone git@github.com:Oyola345/TP-2-Sem-Python.git
   cd TP-2-Sem-Python
   ```

2. Crear el entorno virtual para el proyecto:
    ```
    python -m venv venv
    ```

3. Activar dicho entorno virtual:
    ```bash
    #Windows (usando PowerShell):
    .venv\Scripts\Activate.ps1

    #Windows (usando CMD):
    .venv\Scripts\Activate.bat

    #Linux/MacOs:
    source venv/bin/activate
    ```

4. Instalar Jupyter Lab:
    ```
    pip install jupyterlab
    ```

## 🚀 Ejecución
*Nota: Siempre previamente activar el entorno virtual con el comando del paso 3 de la instalación*

1. Ejecutar el jupyter lab en la carpeta del proyecto:

```
jupyter lab
```

2. En el navegador se abrirá una dirección de localhost donde verá las carpetas notebooks y src.
Haga doble click en la carpeta notebooks y allí habrá dos notebooks para ejecutar:
    * `notebook.ipynb` → Ejercicios 1 al 6
    * `notebook ejercicio_10.ipynb` → Ejercicio 10 "Competencia de cocina"

_**IMPORTANTE:** No cerrar la terminal mientras se trabaje con el proyecto._
